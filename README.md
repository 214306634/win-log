双源加载：支持读取本机日志（System/Security/Application）和导入外部 .evtx 文件。

时间轴查询：根据自定义时间范围重新加载日志，精准定位案发时间点。

深度行为分析：自动提取 登录类型（远程桌面/网络/交互等）、源IP、账户名 等关键字段。

上下文信息：展示进程创建（4688）、服务安装（4697）等事件的命令行、父子进程信息。

智能筛选：支持事件分类、自定义ID、关键词、登录类型、源IP组合筛选。

一键导出：将筛选结果导出为 JSON 格式，便于报告与归档。

🚀 快速开始
1. 环境要求
Windows 7 / 10 / 11 / Server 2016+

Python 3.7+

2. 安装依赖
bash
pip install -r requirements.txt
3. 运行工具
右键点击 main.py，选择“以管理员身份运行”（读取安全日志必须）。

bash
python main.py
📦 依赖项
PyQt5 >= 5.15.0 — 图形界面

python-evtx >= 0.7.4 — 解析 .evtx 文件

pywin32 >= 306 — 调用 Windows 事件日志 API

🛠️ 打包为独立 EXE
bash
pip install pyinstaller
pyinstaller -F -w --uac-admin --name LogAnalyzer main.py
📖 使用指南
加载日志：选择“系统/安全/应用程序”日志类型，点击“加载日志”（默认最近24小时）。

时间查询：在筛选面板调整起止时间，点击“查询”重新加载该时间段日志。

精确筛选：

登录类型：选择“10-远程桌面”可快速定位 RDP 登录。

源IP：输入可疑 IP 进行过滤。

事件分类：按“开关机”、“用户登录”等分组筛选。

查看详情：点击表格任意行，右侧详情区会显示完整事件消息及提取的额外信息（登录类型、源IP、命令行等）。
